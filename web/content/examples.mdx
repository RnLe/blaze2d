# Blaze Examples

Blaze is a Rust Maxwell solver with a Python interface designed for quick experiments and reproducible research. For simple work, the high-level `blaze.solve(...)` function gives you band diagrams directly in Python. For serious sweeps, shared configurations, and repeatable runs, Blaze uses TOML files through the CLI and `BulkDriver`.

The same split appears in the examples below. The visible snippets are Python-first because that is the fastest way to get started, while every browser run is executed through the aligned WASM bulk driver using the same Maxwell configuration shape that the CLI and Python `BulkDriver` consume.

| Interface | Best for |
| --- | --- |
| `blaze.solve(...)` | Fast notebooks and one-off band diagrams |
| CLI flags | Quick shell runs and scripts |
| TOML with `BulkDriver` | Reproducible simulations, sweeps, selective output, and version-controlled configs |

  <ExampleCard
    id="streaming"
    category="Streaming"
    title="Stream Jobs as They Finish"
    description="Use TOML directly when the configuration is the artifact you want to share and reproduce."
    tags={["BulkDriver", "TOML", "streaming"]}
    toml={EXAMPLE_CONFIGS.streaming}
    mode="job-stream"
  >

```python
from blaze import BulkDriver

config_toml = """
polarization = "TM"

[bulk]
threads = 4

[solver]
type = "maxwell"

[geometry]
eps_bg = 12.0

[geometry.lattice]
type = "square"
a = 1.0

[[geometry.atoms]]
pos = [0.5, 0.5]
radius = 0.2
eps_inside = 1.0

[grid]
nx = 20
ny = 20
lx = 1.0
ly = 1.0

[path]
preset = "square"
segments_per_leg = 4

[eigensolver]
n_bands = 4
tol = 1e-5
max_iter = 100

[[sweeps]]
parameter = "atom0.radius"
values = [0.18, 0.24, 0.30]
"""

driver = BulkDriver.from_toml(config_toml, threads=4)

for result in driver.run_streaming():
    print(result["sweep_values"], result["bands"][0][:3])
```

    description="Filter the returned data when you only need a few symmetry points or a small band subset."

```python
from blaze import BulkDriver

config_toml = """
polarization = "TM"

[bulk]
threads = 1

[solver]
type = "maxwell"

[geometry]
eps_bg = 12.0

[geometry.lattice]
type = "square"
a = 1.0

[[geometry.atoms]]
pos = [0.5, 0.5]
radius = 0.22
eps_inside = 1.0

[grid]
nx = 24
ny = 24
lx = 1.0
ly = 1.0

[path]
preset = "square"
segments_per_leg = 5

[eigensolver]
n_bands = 6
tol = 1e-5
max_iter = 120
"""

driver = BulkDriver.from_toml(config_toml)
results, stats = driver.run_collect(
    k_indices=[0, 5, 10, 15],
    band_indices=[0, 1, 2],
)

result = results[0]
print(result["num_k_points"], result["num_bands"])
```

```python
[path]
preset = "triangular"

[eigensolver]
n_bands = 5
tol = 1e-5
max_iter = 120
"""
```
